# Frontend Technical Task

Install apps:
- git clone https://github.com/arwinapel/recipe.git
- cd recipe
- npm install

Setup apps:
- go to `/path/to/recipe`
- npm run mock:api
- open new `terminal` or windows `cmd`
- npm start

For test:
- go to `/path/to/recipe`
- npm test

For Build apps:
- please setup `AppConst.js` first and change URL
